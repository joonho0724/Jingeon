# 경기장 좌표 수동 확인 방법

카카오맵 웹사이트에서 직접 좌표를 확인하는 방법입니다.

## 방법 1: 카카오맵 웹사이트에서 확인

1. **카카오맵 접속**: https://map.kakao.com

2. **주소 검색**: 검색창에 경기장 주소 입력
   - 예: "제주특별자치도 서귀포시 서홍동 477-1"

3. **위치 클릭**: 검색 결과에서 해당 위치 클릭

4. **좌표 확인**: 
   - 방법 A: 브라우저 주소창 URL 확인
     - URL에 `lng=경도&lat=위도` 형식으로 포함될 수 있음
   - 방법 B: 개발자 도구 사용
     1. F12 키를 눌러 개발자 도구 열기
     2. Console 탭 클릭
     3. 다음 코드 입력:
     ```javascript
     // 지도 객체 확인 (카카오맵이 로드된 후)
     // 지도에서 위치를 클릭하면 좌표가 표시됨
     ```

## 방법 2: 카카오맵 모바일 앱 사용

1. 카카오맵 앱에서 주소 검색
2. 위치를 길게 누르면 좌표가 표시됨
3. 또는 공유하기 → 위치 정보에서 좌표 확인

## 방법 3: 다른 지도 서비스 활용

### 네이버 지도
1. https://map.naver.com 접속
2. 주소 검색
3. 위치 클릭 후 우클릭 → "이곳의 좌표 복사"
4. 복사된 좌표는 네이버 좌표계이므로 카카오맵 좌표로 변환 필요

### 구글 지도
1. https://maps.google.com 접속
2. 주소 검색
3. 위치 클릭 후 우클릭 → "이곳의 좌표"
4. WGS84 좌표계이므로 카카오맵 좌표로 변환 필요

## 추천 방법

**가장 정확한 방법**: 카카오맵 웹사이트에서 직접 확인
1. https://map.kakao.com 접속
2. 주소 검색
3. 위치 클릭
4. 브라우저 주소창에서 URL 확인하거나, 개발자 도구 Network 탭에서 API 호출 확인

## 좌표 형식

카카오맵은 WGS84 좌표계를 사용합니다:
- **위도 (lat)**: 33.xxxx (제주도는 약 33도)
- **경도 (lng)**: 126.xxxx (제주도는 약 126도)
