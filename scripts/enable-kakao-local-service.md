# 카카오맵 "로컬" 서비스 활성화 방법

## 현재 상태
✅ REST API 키: 정상 인식됨
❌ 로컬 서비스: 비활성화됨

## 해결 방법

### 1. 카카오 개발자 콘솔 접속
1. [카카오 개발자 콘솔](https://developers.kakao.com) 접속
2. 로그인
3. 내 애플리케이션 선택

### 2. 제품 설정에서 "로컬" 서비스 활성화
1. 왼쪽 메뉴에서 **제품 설정** 클릭
2. **로컬** 서비스 찾기
3. **로컬** 서비스의 **활성화 설정** 버튼 클릭
4. 약관 동의 및 서비스 활성화

### 3. 확인
- "로컬" 서비스가 **활성화** 상태로 변경되었는지 확인
- 상태가 "활성화"로 표시되면 완료

### 4. 다시 테스트
로컬 서비스를 활성화한 후, 다시 스크립트를 실행하면 정상 작동합니다.

## 참고
- 로컬 서비스는 무료로 제공됩니다
- 활성화 후 즉시 사용 가능합니다
- 별도의 승인 절차는 없습니다
