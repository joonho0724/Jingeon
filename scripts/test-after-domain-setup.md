# 도메인 등록 후 확인 방법

## 확인 단계

### 1. 브라우저 캐시 삭제
- **Chrome/Edge**: `Ctrl + Shift + Delete`
- 또는 하드 새로고침: `Ctrl + Shift + R`

### 2. 페이지 새로고침
- `/venues` 페이지로 이동
- `Ctrl + Shift + R` 또는 `Ctrl + F5`로 하드 새로고침

### 3. 개발자 도구에서 확인
- F12 → Network 탭
- `sdk.js` 요청 확인
- **Status 코드 확인**:
  - ✅ **200 OK**: 정상 작동
  - ❌ **401 Unauthorized**: 여전히 인증 문제 (도메인 설정 재확인 필요)
  - ❌ **403 Forbidden**: 다른 권한 문제

### 4. 콘솔 확인
- F12 → Console 탭
- `[VenueMap]` 로그 확인
- 에러 메시지 확인

## 예상 결과

도메인이 제대로 등록되었다면:
- Network 탭에서 `sdk.js` 요청이 **200 OK**로 성공
- 콘솔에 `[VenueMap] 카카오맵 스크립트 로드 성공` 메시지
- 지도가 정상적으로 표시됨

## 문제가 계속되면

1. **도메인 설정 재확인**:
   - Web 플랫폼: `http://localhost:3000`
   - JavaScript SDK 도메인: `localhost:3000` 또는 `localhost:*`

2. **저장 확인**:
   - 플랫폼 설정 변경 후 **저장** 버튼 클릭했는지 확인

3. **브라우저 완전 재시작**:
   - 브라우저를 완전히 종료 후 다시 실행

4. **시크릿 모드 테스트**:
   - 시크릿 모드에서 `/venues` 페이지 접속하여 캐시 영향 제거
