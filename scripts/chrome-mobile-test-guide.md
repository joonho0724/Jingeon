# 크롬 브라우저 모바일 테스트 방법

## 방법 1: 개발자 도구 디바이스 모드 (가장 쉬움)

### 단계
1. **F12** 키를 눌러 개발자 도구 열기
2. 개발자 도구 상단의 **디바이스 모드 아이콘** 클릭
   - 또는 `Ctrl + Shift + M` (Windows/Linux)
   - 또는 `Cmd + Shift + M` (Mac)
3. 상단에 디바이스 선택 바가 나타남
4. 원하는 디바이스 선택:
   - **iPhone 12 Pro**
   - **iPhone SE**
   - **Samsung Galaxy S20**
   - **iPad Pro**
   - 또는 **Responsive** (임의 크기 조정)

### 추가 설정
- **크기 조정**: 상단 바에서 직접 픽셀 크기 입력 가능
- **회전**: 회전 아이콘으로 가로/세로 모드 전환
- **네트워크 제한**: 상단 바에서 네트워크 속도 시뮬레이션 가능

## 방법 2: 수동으로 화면 크기 조정

1. **F12** 키로 개발자 도구 열기
2. 개발자 도구를 별도 창으로 분리 (옵션)
3. 브라우저 창 크기를 직접 조정
4. 개발자 도구에서 현재 화면 크기 확인

## 방법 3: 실제 모바일 디바이스에서 테스트

### Chrome Remote Debugging
1. 모바일 기기와 PC가 같은 Wi-Fi에 연결
2. 모바일 Chrome에서 `chrome://inspect` 접속
3. PC Chrome에서 `chrome://inspect` 접속
4. 모바일 기기에서 웹사이트 접속
5. PC에서 모바일 기기의 페이지를 원격으로 디버깅

## 경기장 정보 페이지 테스트 시 확인 사항

### 모바일 모드에서 확인할 것들:
1. **지도 표시**: 지도가 모바일 화면에 맞게 표시되는지
2. **경기장 목록**: 경기장 카드가 모바일에서 먼저 표시되는지
3. **길찾기 버튼**: 각 경기장 카드에 "길찾기" 버튼이 보이는지
4. **길찾기 기능**: 버튼 클릭 시 카카오맵으로 연결되는지
5. **터치 제스처**: 지도 확대/축소/이동이 터치로 작동하는지

## 빠른 테스트 단축키

- **디바이스 모드 토글**: `Ctrl + Shift + M` (Windows/Linux)
- **개발자 도구 열기/닫기**: `F12`
- **페이지 새로고침**: `Ctrl + R` (일반) / `Ctrl + Shift + R` (하드 리프레시)
